# Emotion Detection ML Project

## Project Overview
This project implements Multi-Class Emotion Detection from Twitter text using Machine Learning techniques and TF-IDF feature extraction.

## Dataset
- tweet_emotions.csv
- Twitter-based emotion dataset

## Emotion Classes
- Happiness
- Sadness
- Worry
- Neutral
- Love
- Surprise

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- NLTK

## Machine Learning Models
- Naive Bayes
- Support Vector Machine (SVM)
- Logistic Regression

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC Curve
- AUC Score

## Final Results

| Model | Accuracy |
|---|---|
| Naive Bayes | 39.10% |
| SVM | 39.32% |
| Logistic Regression | 42.01% |
